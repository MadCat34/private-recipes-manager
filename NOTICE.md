## Modifications

This fork adds:

- GitLab (SaaS and self-hosted) repository support
- BitBucket repository support
- Private Packagist support
- GitHub Package Registry support
- GitLab Package Registry support
- JFrog Artifactory registry support
- Satis registry support
- JSON Schema validation for manifest files
- Configurable recipes repository to enable / disable checks
- New commands to simplify CI integration (files linting, formatting, ...)
- Mkdoc-generated documentation

This fork removes:

- Support for Symfony official recipes repository
- Support for Symfony contrib recipes repository
